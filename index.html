Java

Java is a popular programming language for coding web applications. After more than 20 years of popularity among developers, there are millions of Java applications in use today. Java is a multi-platform, object-oriented and network-centric language that can be used as a platform in its own right. A fast, secure, and reliable programming language for coding everything from mobile apps and enterprise software to big data applications and server-side technologies.

Detail

  • History

    Originally, it was intended to extend C++ with the goal of making software to be installed in home appliances, but Realizing that C++ has limitations, it was created after trying to compensate for the shortcomings of C++.


  • Characteristics

    1.In any environment, if it is written in Java code, it can be executed in the same way.

    2.It is an object-oriented language.

    3.Java manages memory more efficiently than C language.

    4.'Multi-threading' is supported.
